# random-password-generator
This is a command line customizable Password Generator built using Python. The program allows users to generate secure random passwords by selecting different character types such as lowercase letters, uppercase letters, numbers, and special symbols. Users can also specify the desired password length. The generator ensures that the password includes the selected character types and randomizes the final result for better security. This project demonstrates the use of Python modules such as random and string, along with concepts like functions, conditional logic, user input handling, and list manipulation.
